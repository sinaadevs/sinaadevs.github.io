02 — AI
Modular AI Framework

A reusable AI engine built on behavior trees and finite state machines. Features vision cone and hearing-based perception, a memory system with threat levels that escalate over repeated detections, Roblox PathfindingService integration with stuck detection, randomized patrol routing, and squad AI where one NPC spotting you alerts nearby allies. Includes a real-time debug overlay showing NPC state, vision cones, hearing radius, and target lines.

Behavior Trees State Machines Perception System Memory + Threat Pathfinding Squad AI Debug Tools
03 — Systems
Replay + Killcam System

A snapshot-based replay engine recording every player's position 20 times per second in a rolling buffer. On death, the victim sees a cinematic killcam showing the attacker's last 3 seconds of movement as an interpolated ghost model with a follow camera. Includes spectator mode with player switching, a full round system with lobby/battle/finishing states, sword distribution on battle start, and automatic camera and state restoration between rounds.

Snapshot Recording Killcam Interpolation Spectator Mode Round System Cinematic Camera Ghost Playback
04 — Economy
Inventory + Shop System

A fully server-validated inventory and shop system. Items are defined in a single config file with categories, stack limits, buy and sell prices. The server owns all transaction logic — clients can never award themselves items or currency. Features equip/unequip with category-slot logic, a scrollable inventory grid UI, a shop panel, sound effects on all interactions, and clean separation between InventoryService, ShopService, and their client controllers.

Server Validation Anti-Exploit Item Config Equip System Shop UI Inventory UI Sound FX
05 — UI
Cinematic Main Menu System

A production-quality main menu with a cinematic camera system that smoothly transitions between map points, always facing a defined center. Features animated buttons with hover glow, scale tweens and click feedback, a settings panel with draggable volume sliders and toggle switches, an update logs panel, a credits panel, and background music that fades in on load and fades out when the player clicks Play. Player character is hidden and frozen until gameplay begins.
